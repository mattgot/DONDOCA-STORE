# 📝 Sobre

O projeto **E2K Móveis**, é um projeto de fins acadêmicos, com o intuíto de realizar uma atividade extensiva, com a intenção de ajudar uma empresa a ter visibilidade em seu negócio e também a atuar no ramo tecnológico, tendo a facilidade do vínculo mercantil na palma de sua mão. 

# 🤖 Tecnologias Utilizadas

O projeto será desenvolvido utilizando as seguintes tecnologias: 

- ReactJS
- Node
- JavaScript

# 📁 Onde baixar o projeto

```bash

    # Clonar o repositório
$ git clone https://github.com/mattgot/E2K-M-VEIS
    # Entrar no diretório
$ cd E2K-M-VEIS
    # Instalar as dependências
$ yanr install
    # Iniciar o projeto
$ yarn start
